# Google Meet Auto-Add

**Never forget to add video conferencing to your Google Calendar events again!**

A Chrome extension that adds a convenient "Make it a Google Meet" button to Google Calendar, making it easy to add video conferencing to any event with a single click.

## Features

- ✅ **One-click Google Meet integration** - Add video conferencing instantly
- ✅ **Works with new and existing events** - Functions in both creation and editing modes
- ✅ **Seamless integration** - Matches Google Calendar's native design
- ✅ **Smart detection** - Only appears when video conferencing isn't already added
- ✅ **Automatic saving** - Adds Meet link and saves the event automatically

## Installation

### From Chrome Web Store (Recommended)

1. Visit the [Chrome Web Store listing](#) (link coming soon)
2. Click "Add to Chrome"
3. Confirm the installation

### Manual Installation (Development)

1. Download or clone this repository
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode" (toggle in top-right)
4. Click "Load unpacked" and select the extension folder

## Usage

1. Go to [Google Calendar](https://calendar.google.com)
2. Create or edit any event
3. Click the "Make it a Google Meet" button next to the Save button
4. The extension will automatically:
   - Add a Google Meet link to your event
   - Save the event
   - Show a success confirmation

## Privacy

This extension:
- ✅ Only runs on calendar.google.com
- ✅ Does not collect any personal data
- ✅ Does not send data to external servers
- ✅ Only accesses the current tab when you use it
- ✅ Is completely open source
